# Exemplo de Diretiva

Esta é uma diretiva de exemplo que demonstra como estruturar uma SOP (Standard Operating Procedure).

## Objetivo
Descreva claramente o que esta diretiva pretende alcançar.

## Entradas
- Lista de entradas necessárias
- Formato esperado dos dados
- Pré-requisitos

## Ferramentas/Scripts
- `execution/script_exemplo.py` - Script que executa a tarefa principal

## Processo
1. Passo 1: Descrição do que fazer
2. Passo 2: Próxima ação
3. Passo 3: Validação

## Saídas
- Formato esperado da saída
- Onde os resultados serão salvos
- Formato de entrega

## Edge Cases
- Casos especiais a considerar
- Tratamento de erros
- Limitações conhecidas
